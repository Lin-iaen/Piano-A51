PC_piano 项目说明

1. 项目目标
- 基于 8051 的电子琴控制程序。
- 支持串口、按键、红外三路输入。
- 主循环使用统一事件分发，驱动蜂鸣器与 LED 频谱显示。

2. 目录结构
- App: 业务调度与协议解析。
- Drv: 定时器、串口、红外、GPIO 驱动。
- Bsp: 蜂鸣器、按键、LED 板级支持。
- Lib: 延时与基础算法。
- Readme: 文档与问题记录。

3. 事件约定
- 0: 无事件
- 1..21: 发音事件
- 22: 静音事件

4. 构建说明
- 使用 Keil C51 打开 PC_piano.uvproj。
- 目标芯片按工程配置执行 Build。

5. 维护建议
- 新输入源统一输出事件码，不直接调用 Buzzer_Play。
- 中断与主循环共享数据时，先做原子快照再清就绪标志。
- 变更协议后同步更新解析与测试用例。
