1. Lib -- 纯逻辑层 —— 熟悉基本指令与堆栈
    重点在函数开头的 PUSH 和结尾的 POP。在汇编里，寄存器 R0-R7 是公共财产 ，在函数里要用它们，就必须先 PUSH 存进堆栈（压入），用完后再 POP 还回去（弹出），而且顺序必须是先进后出（镜像对称）

2. Drv 裸机驱动层 —— 掌控硬件寄存器与中断
    SFR (特殊功能寄存器) 操作： 重点在于怎么用 ANL (与)、ORL (或) 来配置 TMOD 和 TCON
    看 UART_ISR 和 Timer0_ISR。重点理解为什么中断函数以 RETI 结尾（而不是 RET）。学习环形缓冲区是怎么用汇编里的间接寻址 MOV @R0, A 来存数据的

3. Bsp 板级支持包 —— 查表与硬件映射
    重点在 MOVC A, @A+DPTR 这条指令。这是 51 单片机唯一从 ROM（Flash）里读取表格数据的方法。低、中、高音频率数据靠它取出来

    条件判断： 关注 JZ (如果是 0 就跳转) 和 SUBB (带借位减法) 是怎么配合实现类似 C 语言里 if-else 的。
4. App 业务逻辑层 —— 状态机与整体调度
    多重分支： 汇编里没有 switch-case，全靠 CJNE (比较不相等则跳转) 和 JC (如果有借位/小于则跳转) 来判断用户按下了哪个键

    链接与总控： 看 Main.asm 是如何把分散的文件用 EXTRN 和 LCALL 串联起来，并妥善安放中断向量表（ORG 000BH 等）
