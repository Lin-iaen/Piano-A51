PC 上位机说明

1. 角色定位
- 上位机负责连接管理、指令发送、日志观察与联调支持。
- 单片机负责实时执行与硬件闭环控制。

2. 最小功能
- 串口连接与断开。
- 音符发送(note 0..21)。
- 收发报文日志显示。
- 异常提示与重连。

3. 协议建议
- 帧格式: Header(0x55AA) + Ver + Cmd + Len + Payload + CRC16。
- 命令: Cmd=0x02，payload[0] 为音符值。
- 音符范围: 0=静音，1..21=发音。

4. 联调要求
- 串口参数与固件配置一致。
- 发送非法音符时，上位机提示并阻止发送。
- 长时间发送时记录错误统计与时间戳。

5. 验证清单
- 连续 10 秒按键发送无明显漏发。
- 停止发送后设备可稳定进入静音。
- 断开串口后可恢复连接并继续控制。
